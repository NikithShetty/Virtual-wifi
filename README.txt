Before Starting/Running the program, set it to "Run as Administrator"
	Open properties of the program/Virtual wifi.exe
	Switch to Compatibility tab
	Under privilege Level, check to Run as Adminstrator"


When the network is created, to share  internet through Virtual Wifi while connected to other WiFI source
	Contol panel->Network and sharing->Change Adapter settings->Wireless network connection
	In sharing tab check "Allow other network users to connect through the computer connection"
	select Virtual wifi in Home network connection


When the network is created, to share  internet through Virtual Wifi while connected to internet through Wired cables(Local Area Connection)
	Contol panel->Network and sharing->Change Adapter settings->Local Area Connection
	In sharing tab check "Allow other network users to connect through the computer connection"
	select Virtual wifi in Home network connection
